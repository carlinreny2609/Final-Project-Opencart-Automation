**Project Name: Automated Test Scripts for Open Cart Website**

OpenCart is a free open source e-commerce platform for online merchants. 
This project aims to build a automated testing Solution using Selenium and TestNG automation,
improving efficiency and reducing repetitive code.

Demo Site URL: https://awesomeqa.com/ui/index.php?route=common/home

**Test Scope**
**Following Features Automated**

- User Registration
- Login
- Product Search
- Product Comparison
- Add to Wishlist
- Order Placement
- Browsers Covered: Chrome,Edge
  
**Resources**

- Tools and technologies:
- Tools: Java, Selenium, GitHub,TestNG,Eclipse
- Team: 1 tester, 1 reviewer
